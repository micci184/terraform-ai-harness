# Fargate Public API Example

This example is a reference implementation pattern for the Terraform AI Harness.

It assumes:

- 1 AWS account
- Primary Region: ap-northeast-1
- Public access
- CloudFront + CloudFront WAF
- ALB + ECS Fargate
- Aurora

This directory is for reference only.

Do not treat this example as the default apply target.
Actual Terraform implementation should be placed under `terraform/`.
